У вас есть файлы:
1. Пример Исходник
2. Пример Преобразование
3. Пример Итог
4. Задание Исходник
5. Задание Итог

Файл "Пример Преобразование" написан на языке xslt, он преобразует файл "Пример Исходник" в файл "Пример Итог".
Необходимо на языке xslt написать преобразование для файла "Задание Исходник", чтобы получился файл "Задание Итог".
